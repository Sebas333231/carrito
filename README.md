# carrito
Este es un proyecto basado en un carrito de compras, los productos estan en cards en las cuales se dice el nombre, precio y dos botones los cuales uno de ellos el cual es el carrito sirve para seleccionar el producto y mandar los datos de dicho producto al carrito y ponerlo en una factura donde puedes escoger la cantidad dependiendo la cantidad de los productos se vam haciendo las ecuanciones correspondientes para dar un sub total y total, ademas en el boton que trae la card el cual se llama "informacion", al precionar muestra una modal con los ingredientes del producto, una imagen y nombre.
